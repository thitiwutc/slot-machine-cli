# slot-machine-cli

Play a slot machine in your terminal.

## Usage

```shell
$ slot -h
Usage: slot [options]

Options:

  -h	Display the help message
  -nreel int
    	Number of reels to spin (default 3)
  -nspin int
    	Number of times the slot matchine spins (default 1)
  -symbols string
    	Symbol set in reel (default "🍒🍋🍊🍇🍉🐶🐱🦆🦓🐟🍀💎🔔")
```
